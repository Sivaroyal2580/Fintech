
Description:Fintech mobile app for finance operations.

Requirements
NodeJS v10.16.1;
Yarn v1.17.3;
CocoaPods;
React v16.8.6;
React Native v0.60.5;
xstudio.
android studio.
pre configured simulator/emulator.


How to run locally
Navigate to the project folder: cd ./fintech;
Install project dependencies: npm install;
Install CocoaPods iOS packages: cd ios && pod install;
Run in iOS Simulator: npx react-native run-android;
Run in Android Simulator: npx react-native run-ios ;
